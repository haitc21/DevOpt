# Tổng quan

## 1. Mô tả các tính năng chính của Kibana

| STT | Menu Cha | Menu Con | Mô Tả |
|-----|----------|----------|------|
| 1 | Analytics | | Nhóm tính năng phân tích & trực quan dữ liệu |
| 1.0 | Analytics | Home | Trang tổng quan chung của Kibana |
| 1.1 | Analytics | Discover | Truy vấn & phân tích dữ liệu thời gian thực. Đây cũng là nơi khởi đầu để tạo các Dashboard (Cần tạo Data View trước) |
| 1.2 | Analytics | Dashboard | Tổng hợp biểu đồ dữ liệu trực quan |
| 1.3 | Analytics | Maps | Trực quan dữ liệu theo vị trí địa lý |
| 1.4 | Analytics | Machine Learning | Phân tích nâng cao & phát hiện bất thường |
| 1.5 | Analytics | Visualize Library | Quản lý và tái sử dụng biểu đồ |
| 2 | Elasticsearch | | Nhóm quản lý trực tiếp Elasticsearch |
| 2.1 | Elasticsearch | Home | Tổng quan cluster và tài nguyên |
| 2.2 | Elasticsearch | Index Management | Tạo / quản lý index, lifecycle, shard |
| 2.3 | Elasticsearch | Playground | Test & phân tích truy vấn Elasticsearch |
| 2.4 | Elasticsearch | Synonyms | Quản lý synonyms phục vụ tìm kiếm |
| 2.5 | Elasticsearch | Query rules | Thiết lập quy tắc ưu tiên tìm kiếm |
| 3 | Observability | | Quan sát logs / metrics / uptime / APM |
| 3.1 | Observability | Logs | Theo dõi log tập trung |
| 3.2 | Observability | Metrics | Giám sát tài nguyên hệ thống |
| 3.3 | Observability | APM | Giám sát hiệu năng ứng dụng |
| 3.4 | Observability | Uptime | Theo dõi trạng thái dịch vụ |
| 3.5 | Observability | Alerts & Rules | Thiết lập và quản lý cảnh báo |
| 4 | Security (SIEM) | | Giám sát & điều tra sự cố bảo mật |
| 4.1 | Security | Overview | Tổng quan bảo mật hệ thống |
| 4.2 | Security | Alerts | Quản lý cảnh báo bảo mật |
| 4.3 | Security | Timelines | Điều tra sự kiện theo dòng thời gian |
| 4.4 | Security | Cases | Quản lý và xử lý incident bảo mật |
| 4.5 | Security | Hosts | Phân tích theo từng host |
| 4.6 | Security | Network | Phân tích sự kiện mạng |
| 5 | Stack Monitoring | | Giám sát Elastic Stack |
| 5.1 | Stack Monitoring | Elasticsearch | Theo dõi hiệu năng cluster |
| 5.2 | Stack Monitoring | Kibana | Theo dõi instance Kibana |
| 5.3 | Stack Monitoring | Logstash | Giám sát pipeline ETL |
| 5.4 | Stack Monitoring | Beats | Theo dõi agent thu thập dữ liệu |
| 6 | Management | | Quản trị dữ liệu & hệ thống |
| 6.1 | Management | Data Views | Cấu hình ánh xạ dữ liệu hiển thị |
| 6.2 | Management | Snapshot & Restore | Sao lưu và phục hồi dữ liệu |
| 6.3 | Management | Users & Roles | Quản lý phân quyền người dùng |
| 6.4 | Management | Spaces | Phân tách workspace |
| 6.5 | Management | Dev Tools | Console API trực tiếp |
